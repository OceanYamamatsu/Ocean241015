

```mermaid
sequenceDiagram
%% 病欠届提出のシーケンス図
actor d as 医師
actor s as 学生
actor o as 事務局
actor t as 教員
participant db as 出席データベース

s ->>+ d : 診断書の発行を依頼
d -->>- s : 診断書を発行・受け渡し
loop
    s ->>+ o : 診断書を提出
    o ->>+ t : 病欠通知
    t ->>+ db : 「病欠」を登録
    db -->>- t : 登録完了
    opt
    t -->>- o : 完了通知
    o -->>- s : 完了通知
    end
end
```
